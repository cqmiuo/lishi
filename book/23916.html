内容简介：

本套書的宗旨是希望能夠讓讀者透過實務操作來充分的瞭解Windows Server 2012，進而能夠輕鬆的控管Windows Server 2012的網路環境，因此，書中不但理論解說清楚，而且範例充足。對需要參加微軟認證考試的讀者來說，這套書更是不可或缺的實務參考書籍。

本書精采內容包括：

．循序漸進介紹 Windows Server 2012的基本功能：從安裝與升級、網路環境設定、桌面環境設定、建立帳戶到磁碟管理，循序漸進引導讀者輕鬆入門。

．完整涵蓋經典且實用的課題，包括：電源計劃、公用與共用資料夾、遠端桌面、離線檔案、陰影複製等。

．身為IT人員必備的知識與技能，包括：帳戶與權限管理、Active Directory網域建置、Active Directory資源回收筒、檔案與列印伺服器管理、備份與還原、移難排除…等。

．介紹漫遊與強制使用者設定檔、群組原則、磁碟配額…等，讓系統管理員充分控管使用者的工作環境。

．透過EFS加密檔案系統、BitLocker磁碟機加密與BitLocker to Go來強化檔案內容的機密性。

．利用檔案檢測、存放裝置報告、檔案分類管理與拒絕存取時的協助等工具，讓系統管理員更有效率的來管理檔案伺服器內的資源

．從網路管理及網路安全的觀念與基本運作切入，進而說明如何強化安全措施，包括：Windows防火牆、安全性原則、自動更新、WSUS更新程式部署…等。

．深入探討高可用性的關鍵議題，包括：磁碟容錯系統(RAID)、存放空間(Storage Spaces)、DFS分散式檔案系統、iSCSI檔案伺服器容錯移轉叢集。

．導入Windows Server 2012 Hyper-V的虛擬環境，包含虛擬網路、虛擬機器等，因此只要一台實體電腦就可以建立完整的網路學習環境。

．涵蓋*的IPv6通訊協定，讓您進入IPv6的殿堂。

．藉由作者多年的實務經驗，詳細列舉實際操作時的心得和技巧，引導您部署穩定的、全年無休的基本運作環境。
